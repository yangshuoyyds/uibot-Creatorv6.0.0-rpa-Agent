### 2. 剑气代码生成器

**功能模块**：
```python
class JianqiGenerator:
    ├── generate_multi_source_template()      # 多数据源采集模板
    ├── generate_data_compare_template()      # 数据对比模板
    ├── generate_config_driven_template()     # 配置驱动流程模板
    ├── generate_public_block_template()      # 公共函数库模板
    ├── generate_init_module_template()       # 初始化模块模板
    ├── generate_full_project_template()      # 完整项目结构
    │   ├── _generate_flow_file()            # 生成流程文件
    │   ├── _generate_config_ini()           # 生成配置文件
    │   └── _generate_readme()               # 生成README
    └── generate()                            # 统一生成接口
```

**使用方式**：
```bash
# 交互模式
python tools/jianqi-generator.py -i

# 命令行模式
python tools/jianqi-generator.py -t multi_source \
  -p '{"author":"张三","source_name":"南航"}' \
  -o 南航模块.task
```
