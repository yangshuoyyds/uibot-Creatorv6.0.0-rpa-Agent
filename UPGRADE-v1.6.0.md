### 2. RPA项目模板生成器 🔧

**文件**：`tools/project-template-generator.py` (约800行)

**功能特点**：

#### 支持的模板类型

1. **rpa_project** - 完整RPA项目结构
   - 六阶段目录结构
   - 标准文档模板
   - 配置文件
   - README文档

2. **requirement_doc** - 需求文档模板
   - 项目概述
   - 业务流程现状
   - 自动化需求
   - 验收标准

3. **design_doc** - 设计文档模板
   - 架构设计
   - 模块设计
   - 数据设计
   - 异常设计

4. **test_doc** - 测试文档模板
5. **deployment_doc** - 部署文档模板
6. **operation_doc** - 运维文档模板

#### 使用方式

**交互模式**：
```bash
python tools/project-template-generator.py -i
```

**命令行模式**：
```bash
# 生成完整项目
python tools/project-template-generator.py -t rpa_project \
  -p '{"project_name":"我的RPA项目","author":"张三"}' \
  -o MyRPAProject

# 生成需求文档
python tools/project-template-generator.py -t requirement_doc \
  -p '{"project_name":"我的RPA项目"}' \
  -o 需求说明书.md
```

#### 生成的项目结构

