# UIBot 6.0 命令手册 Skill

这是一个基于 UIBot 6.0.0.211215(64位) 命令手册制作的 Claude Code Skill，用于帮助用户快速查询命令、生成代码和解决问题。

## ✨ 功能特性

### 基础功能
- **命令查询**：根据功能描述快速查找对应的 UIBot 命令
- **代码生成**：根据需求自动生成 UIBot 脚本代码
- **参数说明**：详细解释命令的参数、用法和注意事项
- **实战示例**：提供丰富的实际应用场景代码示例

### 进阶功能（v1.1.0）
- **快速索引**：按功能和场景快速定位所需命令
- **代码模板**：提供可直接使用的代码模板库（11个）
- **常见问题**：收录常见问题及解决方案（23+）
- **工具函数**：实用工具函数库（6个）

### 企业级功能（v1.2.0）
- **企业级最佳实践**：基于官方模板的深度分析和实践指南
- **流程设计模式**：6种常用设计模式详解（REFramework、状态机等）
- **配置管理方案**：Config.xlsx 标准结构和使用方法
- **异常处理策略**：三层异常处理架构
- **重试机制**：智能重试策略和实现
- **日志管理**：企业级日志记录方案

## 🚀 安装使用

### 方法1：在 Claude Code 中使用

1. 将整个 `UB-skill` 文件夹复制到 Claude Code 的 skills 目录：
   ```
   C:\Users\<用户名>\.claude\skills\
   ```

2. 在 Claude Code 中调用：
   ```
   /uibot 如何点击按钮
   ```

### 方法2：直接使用文档

直接打开对应的 Markdown 文档查阅：
- `uibot.md` - 命令分类和快速参考
- `commands-reference.md` - 详细命令说明
- `examples.md` - 实战代码示例

1. 使用 `curl` 命令获取官方文档网页内容
2. 解析 HTML 提取命令分类和结构
3. 整理命令列表和参数说明

5. 生成 Markdown 格式的 Skill 文件
